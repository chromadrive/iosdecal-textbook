Assorted Cocoapods

UI

  • Hero: View controller transition animations. Super easy to implement!

  • Chameleon: Customize your app with color themes! Great if you want to have a "dark mode"

  • SkeletonView: An elegent way to show loading and where page elements are

  • FlatUIKit: A collection of flat UI elements

  • Material: A UI theme library based on Google's Material Design

UI Elements

  • SAParallaxViewControllerSwift: A very nice looking parallax-scrolling table view

  • Koloda: Tindr-like cards

  • XLPagerTabStrip: A container view controller that lets you swipe between views

  • MGSwipeTableCell: Swipeable table cells with animations and stuff

  • InteractiveSideMenu: A customizable expanding side view (like in the Spotify and Gmail apps)

  • MJRefresh: Pull-to-Refresh

  • SAHistoryNavigationViewController: A Navigation View Controller, but it can remember history and display it like in the task viewer

  • SimpleAlert: Nice looking alerts that you can put other views in

  • SAConfettiView: Confetti! Completely unnecessary, but fun!

  • WobbleView: Make things bouncy!

Media

  • SDWebImage/Kingfisher: Asynchronous downloading and caching images, great for if you're loading lots of images into views!

  • MWPhotoBrowser: A light-weight image browser

  • Lightbox: A really nice looking lightbox-style image viewer

  • ImageViewer: Twitter-like image viewer

  • ImageDetect: Detect and crop faces, texts, or barcodes

  • VideoSplashKit: Have a video as the background for your login/signup page (like Spotify)

  • AudioKit: A very robust library for audio synthesis, processing, and analysis!

Networking

  • SwiftyJSON: Native Swift's JSON handling is frankly terrible, SwiftyJSON makes it infinitely better

  • OAuth2/OAuthSwift: Swift-based OAuth libraries, can be useful if you're looking for an alternative to Firebase for login

  • Just: A client-side HTTP library that's very similar to python-requests

Other Functionality

  • Charts: A beautiful and easy-to-use charts library!

  • GaugeKit: Contains tools to build custom gauges (like the Activity Rings on the Apple Watch!)

  • Realm: Local data persistance, IMO easier to use than CoreData.

  • CoreDragon: Library for creating Drag n' Drop elements. The documentation is in Obj-C, but should be easy enough to figure out the Swift.

  • Onboard: Library for making those "getting started" slides that every app seems to have these days

  • SwiftPasscodeLock: Add a passcode lock (with Touch ID/Face ID support) to your app for extra security

  • FSCalendar: A pretty robust calendar

  • Timepiece: Makes handling time in Swift actually bearable

  • RichEditorView: Text editor that supports rich text

  • CryptoSwift: Cryptography algorithms written in Swift

  • RandomKit: Random data generation
